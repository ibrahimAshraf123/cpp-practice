# Algorithms and Problem Solving — Level 1

> *Learn how to think before you learn how to code.*

---

## About This Folder

This folder contains my solutions to the **50 problems** from the *Algorithms and Problem Solving – Level 1* course, part of the Programming Advices Roadmap.

This course is intentionally language-free. The focus is entirely on **how to think and analyze problems** — not on syntax or code. Every problem here was solved first using logical reasoning and flowcharts, before any code was written.

This is the step most programmers skip. This course makes sure you don't.

---

## What This Course Teaches

- What an algorithm truly is (beyond the textbook definition)
- How to analyze a problem before attempting to solve it
- How to break complex problems into simple, ordered steps
- How to identify inputs, decisions, processes, and outputs
- How to design solutions using flowcharts
- How to think independently of any programming language

---

## How the Problems Are Structured

Each of the 50 problems targets one thinking skill at a time, solved through:

- Logical analysis
- Step-by-step reasoning
- Flowchart design (visual thinking)

No programming language. No syntax. No shortcuts.

> All problems in this folder are later re-solved in C++ in the next level — once the thinking foundation is solid.

---

## Why This Matters

Most beginners jump straight into writing code. They struggle, get stuck, and don't understand why. The reason is almost always the same: they never learned how to think algorithmically first.

This course fixes that. It builds the mental foundation that makes everything else — C++, data structures, system design — easier and faster to learn.

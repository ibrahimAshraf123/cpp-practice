# Introduction to Programming with C++ — Level 1

> *From thinking to real programming — your first true programming language.*

---

## About This Folder

This folder contains my solutions to the **50 problems** from the *Introduction to Programming with C++ – Level 1* course, part of the Programming Advices Roadmap.

This is where thinking meets real code. After building a solid algorithmic foundation in the previous two courses, this course introduces C++ as a tool to express correct thinking — not as something to memorize blindly.

---

## What This Course Teaches

- What C++ is, where it's used, and why it builds stronger programmers
- How to set up a professional C++ development environment
- Writing correct C++ syntax with full understanding of what it does
- Working with input, output, variables, and data types
- Using operators, conditions, and loops correctly
- Building reusable logic using functions
- Working with arrays, structures, and enums
- Applying algorithmic thinking through real C++ code
